OSPRay

A Ray Tracing based Rendering Engine for High-Fidelity Visualization

Download this project as a .zip file Download this project as a tar.gz file

Coming soon

The purpose of OSPRay is to provide an open, powerful, and easy-to-use rendering library that allows to easily build applications that use ray tracing based rendering for interactive applications (including both surface- and volume-based visualizations). OSPRay is completely CPU-based, and runs on anything from laptops, to workstations, to compute nodes in HPC systems.

OSPRay internally builds on top of @Embree (http://embree.github.io) and @ispc (The Intel(c) SPMD Program Compiler, ISPC), and fully utilizes modern instruction sets like Intel(c) SSE, AVX, AVX2, or Intel(c) Xeon Phi(tm) to achieve high rendering performance.

We are currently putting the final touches on our first public "alpha" release of OSPRay (to be released on this very site). Please hang on.
